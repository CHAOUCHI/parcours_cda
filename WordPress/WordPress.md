# Wordpress - Créer un site web sans coder
Wordpress est un Système de gestion de contenu (CMS). C'est à dire un site web déjà fait que l'on peut installer sur un hergement web pour ensuite ajouter son contenu.

## Le contenu
Les contenus d'un site web sont les informations disponible au utilisateurs de notre site.

- Pages
- Article de blog
- Produit e-commerce

Créer un site wordpress se fait toujours en deux étapes :

1. Mettre en place l'apparence général du site (couleur en-tete)
2. Ajouter du contenu : pages d'acceuil, pages de services, ajouter des produits.

### Se connecter
Pour vous connecter à votre site wordpress il vous faut écrire /wp-admin après le nom de votre site dans la barre de recherche du navigateur.

Pour un site local
```
localhost/wp-admin
```

Pour un site en ligne
```
mon-site.com/wp-admin
```
